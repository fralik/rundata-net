Rundata.info introduction

About

Welcome to the Rundata.info website. It is a web client for the Scandinavian Runic-text Data Base (SRDB). The database contains information about runic inscriptions discovered mainly in the Nordic countries, as well as in places such as Iceland, Scotland, and Russia. This web version is based on the desktop application Rundata. Although the website is called Rundata.info, the project itself is named Rundata-net and is available at http://www.rundata.info. Rundata-net is a close adaptation of the original Rundata. Although the interface is in English, much of the database text and many place descriptions remain in Swedish. If you do not read Swedish, some parts of the content may therefore be difficult to use in full.

Rundata-net was created by Vadim Frolov and Sofia Pereswetoff-Morath.

Rundata.info is designed to run in a modern browser on a desktop computer with a modern display. A horizontal resolution of at least 1920 pixels is recommended for the best experience. The site also works on smartphones and smaller monitors, although the desktop layout remains the primary target.

Access to the Rundata.info website is provided free of charge.

Rundata-net code and support

The source code of Rundata-net is open and distributed under the GNU GPL, which ensures that it remains open. You can obtain the code from the GitHub repository rundata-net. This means you can run Rundata-net in the way that suits you best: on your own machine, on a dedicated server, or in the cloud.
